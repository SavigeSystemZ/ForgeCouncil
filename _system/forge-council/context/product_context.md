# Context: Product

Forge Council is a **software-factory control plane** (Hybrid Build Fabric): artifact-first, policy-gated, memory-backed. Primary value is **organization** of AI-assisted development, not raw chat.

**Users:** solo builders through small teams.  
**Surfaces:** desktop primary, optional web, CLI/API.  
**Execution:** local / self-hosted / cloud runners (uniform contract in later milestones).

See `PRD.md` and `PRODUCT_BRIEF.md` for authoritative scope.
